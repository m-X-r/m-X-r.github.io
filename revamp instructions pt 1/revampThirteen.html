HTML Elements

Now we are going to create our first container, using an HTML Element. An HTML element or container, as I will be referring to them, are defined by a start tag, some content, and an end tag. In essence:

< tagname >...content goes here...< tagname >

Our first "Container" or HTML Element is going to be our html container. This container encompasses everything else. The HTML tag is the largest Russian Doll. It tells the browser that everything inside it makes up the HTML page. In your notepad document go down a space below your declaration and type the following:

<!Doctype html>
<html>


</html>

Dont forget the most important step. Save!

Notice how there is some space between our start tag <html> and our end tag </html> ? That is because we are nesting things, so we need to leave space for all the other containers that need to go in between our html starting and ending tag.
