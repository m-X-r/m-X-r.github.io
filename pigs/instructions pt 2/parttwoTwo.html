Centering Elements

Right now on our page we have a header, our h1 element. We also have paragraphs, our p element, as well as an unordered list that contains links to our other pages. That includes our ul, li, an a elements. Thats important to note, because in CSS we have to name exactly what we want to change.

Open up your threeLittlePigsSyles.css in notepad. Under the body {padding:60px;} type the following:

/*
....................................................
These are notes:

.....................................................
*/

body{

         padding:60px;

}

h1{

         text-align:center;

}

Save your CSS page. Now open any of your game html pages. Since you added a link to your stylesheet in all your pages, they all will get the same change. For me I am going to open up my momPigsHouse.html page. When you open the page up it should look something like this:

See how now our h1 title Mom Pigs House is now in the center of the page instead of lined up to the left side of the page! Check some more pages and you will see now the h1 element on ALL the pages is in the center!

After completing your victory dance lets note a few things about the CSS sheet that we just made. At the top of the CSS page there is this tag.

/* content goes here */

To make notes in CSS (a cascading style sheet) you need to use this tag so that what you type does not affect your code. It is the same as the <!-- content goes here --> tag that you use to make notes in HTML. Each new coding language you learn will have its own set of unique tags to create notes. When learning new langauges be sure to find out what they are for that specific language. I added ................ inside my notes tags just to make the block where I make notes more visible for me. Visibility is the same reason why I have indented padding as well as text-align. I know as I add more items to alter it will be easier for me to keep track of what is what If I add spaces and indents to make things easier for me to see.

Another thing to note is text-align. In html we have our elements like paragraph (p) and header (h1). In CSS we have properties like text-align and font-size. Its hard to remember all the different CSS properties for all HTML elements, but a quick google search (like this one) typically gets you all the info you need.
