Body Container Tags

Now we are gearing up to add a little substance to our body container. But before we do, we need to go over some basic container types to use inside our body container. Below I will show you some basic tags and the code you need to make them happen.

  • H:

    the h or heading tag creates a heading or banner. So, you have text with a space above and below. Headings go from 1-6, one being the largest and six being the smallest. It goes h1, h2, h3, h4, h5 and h6.

    <h1> H1 </h1>

    <h2> H2 </h2>

    <h3> H3 </h3>

    <h4> H4 </h4>

    <h5> H5 </h5>
    <h6> H6 </h6>

  • p:

    the paragraph container, it creates, well, a paragraph. It will block out whatever text you put in with a space above and below. Just like when using the word typing program.

    <p> This is a paragraph. This is a paragraph. This is a paragraph. This is a paragraph. This is a paragraph. This is a paragraph. This is a paragraph. This is a paragraph. This is a paragraph. This is a paragraph. This is a paragraph. This is a paragraph. This is a paragraph. This is a paragraph. </p>

    <p> This is a paragraph. This is a paragraph. This is a paragraph. This is a paragraph. This is a paragraph. This is a paragraph. This is a paragraph. This is a paragraph. This is a paragraph. This is a paragraph. This is a paragraph. This is a paragraph. This is a paragraph. This is a paragraph.</p>

  • ul or ol:

    These are unordered list and ordered list containers. They both do the exact same thing with one little difference. They create a bullet list, ul aka unordered list use round bullet points. Or they create numbered list, ol, aka ordered list use numbers.

    <ul>

    • <li> this </li>
    • <li> is </li>
    • <li> an </li>
    • <li> unordered </li>
    • <li> list </li>

    </ul>


    <ol>

    1. <li> this </li>
    2. <li> is </li>
    3. <li> an </li>
    4. <li> ordered </li>
    5. <li> list </li>

    </ol>

    a:

    The a container holds links to other places. It could be a webpage on the internet or a webpage that you made. You can even go to a different section of the webpage that you are on with the help of the id attribute. The id attribute, like the declaration tag, is less like a container and more like a sign.

    <a href="https://www.richlandlibrary.com"> This is a Link to the Libraries Website </a>
